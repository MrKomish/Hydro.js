it work! bootstrap loaded!

it easy, you write:
hydro(type of file: Css/Js)(link to file in inverted commas('') and in brackets).

if you want many files to load add after inverted commas:
Comma(,) and the link to the file in: inverted commas('').


examples:

 hydroJs(
 'js/bootstrap.min.js'
,'http://code.jquery.com/jquery-1.8.0.js'
,'http://code.jquery.com/jquery-1.7.1.js'
,'http://code.jquery.com/jquery-1.7.2.js'
,'http://code.jquery.com/jquery-1.6.1.js');

See the result in result.js.txt

 hydroCss(
 'css/style1.css'
,'css/style2.css'
,'css/style3.css');


See the result in result.css.txt
